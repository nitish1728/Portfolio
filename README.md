## 💻 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/nitish1728/Portfolio.git
cd Portfolio
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run locally

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view in the browser.

### 4. Build for production

```bash
npm run build
```
